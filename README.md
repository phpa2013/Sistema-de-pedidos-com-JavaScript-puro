# 🍔 Sistema de Lanchonete com JavaScript Puro

Sistema completo de pedidos de lanchonete desenvolvido com JavaScript puro, permitindo ao usuário montar seu pedido com carrinho dinâmico, controle de quantidade e filtros em tempo real.

## 🌐 Acesse o projeto online

👉 https://phpa2013.github.io/Sistema-de-pedidos-com-JavaScript-puro/

---

## 🚀 Funcionalidades

* 📋 Listagem dinâmica de produtos (cardápio)
* 🛒 Adição de itens ao carrinho
* ➕ Aumento de quantidade
* ➖ Diminuição de quantidade
* 💰 Cálculo automático de subtotal
* 💾 Persistência de dados com LocalStorage
* 🔍 Busca por nome
* 🏷️ Filtro por categoria

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES Modules)

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes como:

* Manipulação de DOM
* Event Delegation
* Organização de código em módulos
* Uso de LocalStorage
