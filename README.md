# 🍔 Sistema de Lanchonete

Projeto desenvolvido com JavaScript puro para simular um sistema de pedidos de uma lanchonete.

## 🚀 Funcionalidades

* Listagem de produtos (cardápio)
* Adição de itens ao carrinho
* Aumento e diminuição de quantidade
* Cálculo automático de subtotal
* Persistência com LocalStorage
* Filtro por categoria
* Busca por nome

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (ES Modules)

## 📚 Aprendizados

Neste projeto pratiquei:

* Manipulação de DOM
* Event Delegation
* Organização de código em módulos
* Uso de localStorage
* Métodos de array (map, filter, reduce)

## ▶️ Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador
